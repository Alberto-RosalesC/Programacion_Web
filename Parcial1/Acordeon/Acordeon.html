Las propiedades CSS "float", "overflow" y "clear":
Float (Flotar): float es una propiedad CSS que permite que un elemento se posicione hacia la izquierda o la derecha de su contenedor, permitiendo que otros elementos fluyan alrededor de él.

Overflow (Desbordamiento): overflow es una propiedad CSS que controla el comportamiento del contenido que desborda su contenedor. Puede ocultar el contenido, mostrar barras de desplazamiento o permitir que el contenido desborde.

Clear (Limpiar): clear es una propiedad CSS que se utiliza para controlar el flujo de elementos flotantes en relación con otros elementos en la página. Cuando se aplica, asegura que no haya elementos flotantes en el mismo lado del elemento que se está "limpiando".
