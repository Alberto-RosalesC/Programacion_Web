Pseudo Selectores

Pseudo Elemento Seleccionara:
::first-letterLa primera letra de un elemento de bloque.
::first-lineLa primera linea de un elemento de bloque.
::beforeLa ubicacion que esta antes de un elemento.
::afterLa ubicacion que esta despues de un elemento.
Pseudo Clase Dinamica Seleccionara:
:linkLos enlaces que no se han visitado(Solo para elementos < a >)
:VisitedLos enlaces que ya se han visitado(Solo para elementos < a >)
:hoverElemento cuando el usuario pasa el raton sobre el
:activeElemento activo (Solo durante el click)
:focusElemento que tiene el focus(hasta que lo pierda)
Pseudo Clase Estructural Seleccionara:
:first-chlidEl primer hijo seleccionado.
:last-childEl umtimo hijo seleccionado.
:nth-child(n)Enesimo hijo del elemento seleccionado
:nth-last-child(n)Enesimo hijo seleccionado (empezando del final hacia el inicio)
:first-of-typeEl primer hijo seleccionado.
:last-of-typeEl ultimo hijo del elemento seleccionado del mismo tipo
:nth-of-typeEnesimo hijo del elemento seleccionado del mismo
:nth-last-of-typeEnesimo hijo del elemento seleccionado(empezando del final hacia el inicio)
:only-childSi el elemento tiene un hijo unico
:only-of-typeSi el elemento no tiene hermanos del mismo tipo
:empityLos elementos que no tengan contenido
:rootSiempre es < html >
Pseudo Clases de Interfaz Seleccionara:
:eneableSi el elemento (de formulario) esta habilitado
:disableSi el elemento (de formulario) esta inhabilitado
:checkedSi el elemento checked, radio button y option esta seleccionado
:validLos navegadores proveen validacion basica basica de acuerdo al tipo de elemento < imput >
:invalidValid/invalid se asignan al elemento de acuerdo ala entrada del usuario
:requiredSelecciona elementos que tengan el atributo de required
:optionalSelecciona elementos que no tengan el atributo de required
